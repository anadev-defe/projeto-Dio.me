Tags para criar um Formulário

Topo

Essas tags são mais utilizadas na criação de um formulário. A tag form é utilizada para criar um formulário, porém essa tag sozinha não muda nada na parte estética da página, ela precisa de tags filhas para que haja funcionalidade. Uma tag filha seria o input, ele coloca um campo na tela para o usuário digitar. Porém essa tag é limitada em questão de espaço, caso precise de um campo maior para escrever uma mensagem, utiliza a tag textarea. Existe a tag button que vai criar um botao de "enviar", porém essa tag deve ter um atributo chamado onsubmit para validar o comando de enviar. A tag select serve para colocar um campo de escolha com valores pré definidos, que serve para você colocar cargo, ou função, por exemplo.

Tags de Formatação

Topo

São tags utilizadas para estilização do texto. Algumas dessas tags não são muito utilizadas após a atualização do HTML para HTML5, pois toda essa formatação pode ser feita em um arquivo CSS, deixando o código HTML mais limpo. Já outras tags passaram a ser utilizadas para auxiliar na formatação do texto em CSS. Algumas das tags que caíram em desuso foram strong, b, i, u e font; onde as duas primeiras são para colocar uma palavra em negrito, em seguida em itálico, e a próxima serve para colocar em sublinhado e a última altera a fonte das palavras, é uma tag que precisa de atributos, pois ela sozinha não faz nada.

As tags strong e b tem a mesma finalidade, que é deixar a palavra em negrito, porém a primeira dá mais ênfase no seu texto, importante principalmente para questões de acessibilidade.

Tags de Estruturação

Topo

As principais tags de estruturação são: header, body e footer. O header é popularmente conhecido como cabeçalho, o main é onde fica o assunto principal da página, só pode ser usada apenas uma vez no arquivo HTML, e o footer que é o rodapé da página. Além dessas tags, existe também span e div que são tags coringa para a estruturação do texto. A div separa em quadrantes as áreas da página. Ela é uma tag no formato display block, que ocupa a horizontal da página inteira, já a span não é no formato display block, então ela não ocupa toda a horizontal, permitindo mudar as características do texto de forma idividual pelo CSS. Tem ainda a tag fieldset que serve para delimitar um formulário para carater visual. Existem outras como embed e iframe que não são muito utilizadas, mas a título de curiosidade tem um link no final da página com um vídeo que mostra como usar essas tags.

Tags de Mídia

Topo

Para cada tipo de mídia existe uma tag específica. Para inserir imagens no arquivo HTML é utilizada a tag img, que precisa do atributo de endereço src, é esse atributo que leva ao arquivo salvo na pasta. Essa imagem pode ser salva nos formatos png, jpg e svg, cada formato tem sua particularidade que deve ser levada em consideração quando precisar usar a imagem. A tag audio precisa de uma tag filha source com o atributo src com o endereço do arquivo, também pode ser usado o atributo control que insere o controle de volume, velocidade e tempo de execução do áudio. A tag video tem os mesmos requisitos que a tag audio com relação a tag filha e atributos. O vídeo pode ser salvo nos formatos mp4 e webm.

OBS.: Tanto para as tag audio como para a tag video é importante colocar uma mensagem para o caso do navegador do usuário não ter suporte para abrir a mídia.

Por último a tag track que auxilia a tag video, pois ela insere legendas ou quebra de vídeo.
